# Chaturbate Recorder

Этот скрипт позволяет записывать стримы с платформы Chaturbate в высоком качестве. Он автоматически обновляет ссылку на стрим для непрерывной записи и выбирает лучшее доступное качество видео.

## Функции

- **Автоматическое получение URL стрима**: Через API Chaturbate.
- **Выбор лучшего качества**: Скрипт анализирует HLS плейлист и выбирает вариант с максимальным bandwidth (оригинальное качество).
- **Автообновление ссылки**: Каждые 5 минут проверяет и обновляет URL, перезапуская запись при необходимости для безопасности.
- **Префикс к имени файла**: Включает дату (ДД.ММ.ГГГГ), хеш от имени комнаты (для идентификации) и имя комнаты.
- **Конфигурация**: Имя комнаты передаётся как аргумент командной строки.

## Требования

- Python 3.x
- Библиотека `requests`: `pip install requests`
- FFmpeg: Установите FFmpeg и добавьте в PATH (скачать с https://ffmpeg.org/)

### Установка FFmpeg на Windows

1. Перейдите на https://www.gyan.dev/ffmpeg/builds/ (официальные сборки для Windows).
2. Скачайте последнюю версию ffmpeg-release-essentials.zip (или full).
3. Распакуйте архив в папку, например `C:\ffmpeg`.
4. Добавьте `C:\ffmpeg\bin` в переменную среды PATH:
   - Откройте "Свойства системы" > "Дополнительные параметры системы" > "Переменные среды".
   - Найдите "Path" в "Системные переменные", нажмите "Изменить".
   - Добавьте новый путь: `C:\ffmpeg\bin`.
   - Перезагрузите командную строку или VS Code.
5. Проверьте: откройте командную строку и введите `ffmpeg -version`. Должно показать версию.

Если используете Chocolatey: `choco install ffmpeg`.

## Установка

1. Клонируйте или скачайте репозиторий.
2. Установите зависимости: `pip install requests`
3. Убедитесь, что FFmpeg установлен и доступен в командной строке.

## Использование

Запустите скрипт с именем комнаты в качестве аргумента:

```
python record_chaturbate.py <room_slug>
```

Пример:
```
python record_chaturbate.py seltin_sweety
```

Скрипт начнёт запись в файл с именем вида `25.04.2026_abc12345_seltin_sweety_recording.mp4` (дата, хеш, имя комнаты).

Запись продолжается до тех пор, пока комната активна. Скрипт автоматически обрабатывает обновления стрима.

## Примечания

- Используйте только для личных целей и соблюдайте законы о авторских правах и конфиденциальности.
- Если стрим приватный или комната не существует, скрипт завершится с ошибкой.
- Для остановки записи прервите выполнение скрипта (Ctrl+C).
